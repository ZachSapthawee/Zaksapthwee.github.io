Cascading Style Sheets หรือ CSS เป็นเครื่องมือในการตกแต่งเว็บให้ดูสวยงาม
เพิ่มสีสันหน้าตาของเว็บไซต์ให้ดูดีและเข้าใจง่าย ยิ่งขึ้น จากที่เราจะใช้แค่ PHP เขียนเว็บมันก็จะดูทื่อ เป็นเว็บธรมมดาเราก็จะนำ CSS เข้ามาเพิ่ม UI ให้ดูสวยงามน่าใช้การสรุปนี้เป็นการสรุปตามความเข้าใจของ ทรัพย์ทวี เพ็ชรสาย

CSS Syntax รูปแบบของคำสั่ง CSS 
                            selector{
                                property: value;
                            }
Selector เป็นส่วนชื่อ tag ID หรือ Class ที่ใช้เรียกแสดงผล
Property เป็นส่วนคุณสมบัติในการปรับแต่งที่ใช้ในการแสดงผล เช่น ใส่สี ขนาดตัวอักษร ปรับตำแหน่ง
Value ใช้กำหนดค่าของสิ่งที่ให้แสดงผล เช่น ปรับขนาดสีพื้นหลัง Property = background-color ส่วน Value = #DCDCDC เปลี่ยนสี
